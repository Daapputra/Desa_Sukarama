# Website Resmi Desa Sukarama

Website Resmi Desa Sukarama, Kecamatan Bojongpicung, Kabupaten Cianjur.

## Tentang Proyek

Website ini dibangun untuk menyediakan informasi resmi, layanan, serta potensi UMKM yang ada di Desa Sukarama. 

## Struktur Proyek

- **FE/**: Direktori yang berisi kode *Frontend* (HTML, CSS, JS) dari website.
- **BE/**: Direktori yang berisi kode *Backend* (Node.js/Express) untuk melayani API.

## Prasyarat

Pastikan Anda telah menginstal [Node.js](https://nodejs.org/) di sistem Anda.

## Instalasi

1. Clone repositori ini:
   ```bash
   git clone https://github.com/Daapputra/Desa_Sukarama.git
   ```

2. Masuk ke direktori proyek:
   ```bash
   cd Desa_Sukarama
   ```

3. Instal dependensi:
   ```bash
   npm install
   ```

## Menjalankan Proyek

Untuk menjalankan server di lingkungan pengembangan, gunakan perintah:

```bash
npm start
```
Atau:
```bash
npm run dev
```

Server backend akan berjalan dan siap melayani permintaan dari frontend.

## Teknologi yang Digunakan

- HTML, CSS, JavaScript (Frontend)
- Node.js & Express.js (Backend)
- better-sqlite3 (Database)
- multer (Penanganan Upload File)
- cors
